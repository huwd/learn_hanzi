# Axis Rollup

## Question

How should multiple learning axes (meaning, sound, form, etc.) combine into an overall state for a dictionary entry — if at all?

## Why this matters

Without a rollup, the app can show per-axis strength but can't give a single "where am I with this character?" answer. With a bad rollup, it collapses nuance that the axis model is designed to preserve.
